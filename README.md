<h1>Fullstack Application ~ MongoDB CRUD Using Python with Flask</h1>
<h2>How to use</h2>

$ git clone https://github.com/Xaheersays/FlaskMongoCrud.git

<h2>Install Packages</h2>
$ pip install flask
<hr>
$ pip install pymongo
<hr>
$ pip install flask_pymongo
<hr>
$ pip install Flask-Cors
<hr>
